# Changelog

All notable changes to Cola Records will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-XX

### Added

- **Issue Discovery**: Search GitHub for "good first issues" across repositories
- **Contribution Tracking**: Fork, clone, and track progress on contributions
- **Git Integration**: Built-in Git operations (clone, commit, push, branch, remotes)
- **Embedded IDE**: Code-server (VS Code) running in Docker with full editor capabilities
- **Terminal Integration**: Multi-tab terminal with Git Bash, PowerShell, CMD support
- **Spotify Integration**: Music playback during development sessions
- **Discord Integration**: Community communication client
- **SSH Remotes**: Configure SSH hosts for terminal-based access
- **Pull Request Management**: Create and manage pull requests from within the app
- **Issue Management**: View and interact with repository issues
- **Branch Management**: Create, switch, and manage Git branches
- **Monaco Editor**: Integrated code editor for quick file edits
- **Tools Panel**: Expandable tool box with terminal and editor access
